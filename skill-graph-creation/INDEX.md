# Граф создания графов навыков

Создание графов навыков — это систематический подход к структурированию знаний. Этот мета-граф научит вас создавать, организовывать и поддерживать графы навыков для любой предметной области.

## Процесс создания

Следуйте этому workflow для создания нового графа:

1. [[graph-structure]] — Организация файлов и папок
2. [[content-writing]] — Написание качественных навыков
3. [[linking-strategies]] — Создание связей между навыками
4. [[naming-conventions]] — Правила именования
5. [[validation]] — Проверка качества графа
6. [[integration]] — Подключение к системе

## Быстрый старт

**Для быстрого создания нового графа:**

1. **Выберите тему:** Определите предметную область (медицина, маркетинг, etc.)
2. **Создайте структуру:** Следуйте [[graph-structure]]
3. **Используйте шаблоны:** См. [[content-writing]] для готовых шаблонов
4. **Свяжите навыки:** Примените [[linking-strategies]]
5. **Валидируйте:** Запустите скрипты из [[validation]]
6. **Интегрируйте:** Подключите через [[integration]]

Когда граф готов:

3. Протестируйте агента — задайте вопрос из вашей темы

## Лучшие практики

### Размер

- **Маленький граф:** 5-8 навыков (узкие темы)
- **Средний граф:** 8-15 навыков (широкие темы)
- **Большой граф:** 15+ навыков (комплексные темы)

### Глубина

- **1 уровень:** Основные концепции (каждый новичёк)
- **2 уровня:** Продвинутые темы (практики)
- **3 уровня:** Специфические паттерны (эксперты)

### Связи

- **Средний:** 6-10 ссылок на навык
- **Минимум:** 3-4 связи
- **Максимум:** 15-20 связей (для сложных навыков)

## Частые вопросы

### Какую тему выбрать?

Начните с того, что:
- Вы хорошо понимаете
- Часто помогает другим
- Имеёт достаточно глубины для 5+ навыков

### Сколько навыков создавать?

Начните с 5-7 навыков. Можно всегда добавить позже.

### Как проверить качество?

Используйте автоматические скрипты из [[validation]] + ручную проверку.
